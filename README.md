# Nombre del Proyecto

Desafío de Prueba Técnica

## Tecnologías Usadas

- React
- React Native
- Redux
- React Navigation
- TypeScript
- React Native Paper

## Instalación

1. Clona el repositorio:

   ```bash
   git clone https://github.com/facurueda/Prueba-Tecnica---GTT.git
   cd Prueba-Tecnica---GTT
   ```

2. Instala las dependencias:

   ```bash
   yarn // npm install
   ```

3. Ejecuta el proyecto:

   Para desarrollo web:

   ```bash
   yarn android
   ```
